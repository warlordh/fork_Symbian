RSat::TSetUpMenuRspV1 Class Reference

class RSat::TSetUpMenuRspV1 : public RSat::TSatPCmdBase

Contains your response to a SET UP MENU proactive command. Pass this to the UICC with RSat::TerminalRsp(). See RSat::NotifySetUpMenuPCmd() for more information about the command.

For Symbian OS versions up to and including v8.1a and v8.1b pass RSat::TerminalRsp() a RSat::TSetUpMenuRspV1Pckg (a packaged TSetUpMenuRspV1). For later versions of Symbian OS, check for an updated API (RSat::TSetUpMenuRspV2 etc).
