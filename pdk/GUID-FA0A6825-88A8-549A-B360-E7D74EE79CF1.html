USB Host Mass Storage File System

Provides an FSY plug-in for USB Mass Storage File System. Used to configure connection with attached devices so they can act as a mass storage device.

Introduction

The USB host mass storage file system allows mass storage devices to connect to a Symbian platform device executing as the host.

USB host mass storage features

The USB host mass storage provides the following features:

  • The Symbian platform device acting as host has the ability to access media directly via a USB connection.

  • The device connected to the USB port must be using the FAT file system, otherwise it cannot be mounted.

  • The number of devices that can use this file system is limited to one device and one hub (at the time of writing).
