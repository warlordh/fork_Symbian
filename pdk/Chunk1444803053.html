Event handling

This section describes the different events that a Symbian application can receive from the application framework, focusing in particular on events that are most beneficial for application developers.

The Symbian platform passes user input and system events to applications on the device, which then are responsible for handling the events in the appropriate control.

This section explains the following:

  • Commands
  • Window server events
  • System environment events
  • Domain-specific events
  • Touch-specific events
