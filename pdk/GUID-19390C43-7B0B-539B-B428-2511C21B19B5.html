Packet Data API Tutorial Introduction

This section provides the list of tasks that can be used by the client applications for the packet data service.


  1. To define a packet service, see Packet Service Tutorial for more information.

  2. To define the network interface, see Network Interface Tutorial for more information.

  3. To create a context for a packet service, see Packet Context Tutorial.

  4. To define the context related to Multimedia Broadcast Multicast Service (MBMS), see MBMS Context Tutorial.

  5. To create a Quality of service for a context, see Packet Quality of Service Tutorial.

  6. To create a Traffic Flow Templates for the packet service context, see Traffic Flow Template.

  7. To define the lists to store different types of information related to packet service, see Packet List Tutorial.
