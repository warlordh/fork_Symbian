PointerCursorOnMode()

IMPORT_C TPointerCursorModePointerCursorOnMode()[static]

Gets the mode that should be used when the virtual cursor is switched on.

The return value has no effect for devices not using a virtual cursor.

This function returns an enum of the type TPointerCursorMode, which describes the mode that should be used when the virtual cursor is switched on. The return value has no effect for devices not using a virtual cursor.
