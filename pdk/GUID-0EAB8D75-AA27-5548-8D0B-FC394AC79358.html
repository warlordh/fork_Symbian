cp

cp

The cp command copies files and directories. For more information about using the cp command, search http://www.freebsd.org/cgi/man.cgi by specifying cp as the search keyword and by selecting the FreeBSD 6.2-Release option from the release list.

The following options are not supported by the Symbian version of the cp command:

  • -H

  • -L

  • -P

  • -l

  • -n

  • -v

The following options of the cp command work differently on the Symbian platform.

  • -p: This option is used to copy and preserve the time stamp of a file.

  • -R: This option operates on the contents of directories recursively.
