IMAP Transport Buffer Size

The CImapTransportBufferSizes class is used to store per bearer-type IMAP transport buffer size settings. This class allows configuring the maximum buffer size for data to be transferred, and number of commands that may be simultaneously issued to the remote server (one configuration set for all IMAP accounts).
