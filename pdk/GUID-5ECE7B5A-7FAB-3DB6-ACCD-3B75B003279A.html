HandleResetNetServerReqL()

TInt HandleResetNetServerReqL()[pure virtual]

The CTSY Dispatcher shall invoke this function on receiving the ECustomResetNetServerIPC request from the CTSY.

It is a request call that is completed by invoking CCtsyDispatcherCallback::CallbackPhoneResetNetServerComp().

Implementation of this interface should reset the net server.
