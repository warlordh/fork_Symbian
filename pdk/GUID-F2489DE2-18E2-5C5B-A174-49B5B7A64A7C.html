Feature Manager Reporting Tutorials

This document introduces tutorials on the use of Feature Manager to report on features.

Before you start, you must:

  • Understand what is meant by a feature, feature sets and feature data in Symbian platform.

A feature on Symbian platform encapsulates information such as the presence or absence of some item of hardware or software on the device. These tutorials explain how to use Feature Manager to report on that information.

  • Reporting on Several Features Dynamically Using RFeatureControl
    How to query a feature set using RFeatureControl dynamically.
  • Reporting on all Features on a Device Dynamically Using RFeatureControl
    How to list all supported features on a device using RFeatureControl dynamically.
