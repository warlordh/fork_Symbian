Bluetooth GPS Positioning Module

The Bluetooth GPS Positioning Module (Bluetooth GPS PSY) connects the LBS subsystem to an external Bluetooth GPS device.

  • Bluetooth GPS PSY Overview
    This describes the Bluetooth GPS PSY, which connects the Location Server (eposserver.exe) to external Bluetooth GPS devices. The audience is device creators and developers who want to integrate the Bluetooth GPS PSY into their products.
  • Bluetooth GPS PSY Tutorials
    These tutorials describe how to configure and use the Bluetooth GPS PSY.
  • Bluetooth GPS PSY Settings Storage API Overview
    The Bluetooth GPS PSY Settings Storage API defines Central Repository keys for storing and retrieving the settings of the Bluetooth GPS PSY. The API is for developers who write applications to get and set Bluetooth GPS PSY settings.
  • Bluetooth GPS PSY Events API Overview
    The Bluetooth GPS PSY Events API uses the Publish and Subscribe (P&S) service to notify clients about the state of the PSY and the connected Bluetooth GPS device. The API is for developers who write components to notify users of changes in the Bluetooth GPS PSY state.
  • Bluetooth GPS PSY Configuration API
    The Bluetooth GPS PSY Configuration API enables client applications to manage a list of Bluetooth GPS devices.
  • Bluetooth GPS PSY Reference
    These documents describe NMEA and PNOK messages supported by the Bluetooth GPS PSY.
