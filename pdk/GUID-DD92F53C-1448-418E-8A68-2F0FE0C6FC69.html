Ambient Light Sensor Channel

The ambient light sensor channel indicates the current brightness level.

The following table provides information about the ambient light sensor channel:

Header sensrvilluminationsensor.h
Data type State
Channel type id KSensrvChannelTypeIdAmbientLightData
Data class TSensrvAmbientLightData
Data units 0 to 100 percent light

The ambient light state is measured as a percentage value, where 0 is very dark and 100 is bright sunlight.
