RSat::TMmsNotificationDownloadV6 Class Reference

class RSat::TMmsNotificationDownloadV6 : public RSat::TSatBase

This data structure contains the details of the MMS NOTIFICATION DOWNLOAD envelope command issued to the UICC. See ETSI TS 102 223 and RSat::MmsNotificationDownload() for more information about the command.

Pass a packaged version of the latest "TMMSNotificationDownloadV6" class. For Symbian OS version 9.4 use RSat::TMMSNotificationDownloadV6Pckg, a packaged version of RSat::TMMSNotificationDownloadV6. For later versions of Symbian OS, check for an updated API (RSat::TMMSNotificationDownloadV7 etc).
