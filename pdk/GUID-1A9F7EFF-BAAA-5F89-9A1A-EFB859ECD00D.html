Category: CHWComm

Panic of ECUART module raised by the Serial Comms Server (or "Comms Server").

0

Bad descriptor passed by client.

1

The cancel() method was called without calling the respective API first. An attempt was made to cancel an operation that was not started.

2

Incorrect usage of an API for the current DTE or DCE role.

3

Internal error caused by the notification count being negative.
