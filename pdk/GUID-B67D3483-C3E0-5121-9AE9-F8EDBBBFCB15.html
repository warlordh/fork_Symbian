Reference

This section contains reference information for the components in the Network Control collection.

Quality of Service Ini File Format

File Location

Quality of Service Ini files can be stored in the ROM of the device by placing the file in the \private\101f7989\esock folder. For example z:\private\101f7989\esock\qos.ini.

File Format

The configuration options for the QoS Ini file must be in a section called qosconfig.

Keyword Type Default Value Description

unload-delay

INTEGER

16000000 (Âµs)

Delay for unloading additional QoS modules.

File Format Example

The following example shows an Ini file for the Quality of Service component: 

[qosconfig]
unload-delay= 16000000
