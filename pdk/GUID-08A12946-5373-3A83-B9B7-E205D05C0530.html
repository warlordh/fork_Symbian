const TInt KHCHardResetUid

const TIntKHCHardResetUid

The UID of the Controller Hard Reset API. If the HC Hard Reset API ever has to change, a new UID and associated M- class will be created.

Old (non-updated) Controller Hard Reset plugins will still work as long as the old API is supported
