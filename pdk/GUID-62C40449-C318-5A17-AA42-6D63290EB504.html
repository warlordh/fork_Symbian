Generic Board Support Packages

The kernel side Board Support Packages (BSP) collection contains a comprehensive set of base porting templates.

This document set contains tutorial and guide documents that describe the porting process.

  • Template Variant
    Provides a set of templates for use when porting hardware.
  • Emulator BSP Overview
    The emulator board support package (BSP) provides the variant kernel, executables, debugging framework and device drivers for the Symbian platform (Epoc) emulator.
