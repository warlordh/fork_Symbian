const TInt KGSPluginProviderInternal

const TIntKGSPluginProviderInternal
Private id used for S60 internally provided GS plug-ins. This must not be disclosed to other parties. Using this id allows the plugin to be handled with more rights than other plugins:

  • Inserted before other plugins in listboxes.

  • Can define their order in the listbox.

  • Plug-in can belong also to other views than applications-view.
