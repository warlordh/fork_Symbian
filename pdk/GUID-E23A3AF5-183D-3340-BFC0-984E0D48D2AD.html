TCompensationTypeAxisData Class Reference

class TCompensationTypeAxisData

Definition of content and package for TCompensationTypeAxisData. Applies to TSensrvAccelerometerAxisData and TSensrvMagnetometerAxisData. If multiple compensation items are provided, the items are combined using matrix multiplication in the compensator that requests the items. Due to the multiplication, plugin must provide the items in priority order (i.e. the most relevant compensation first).
