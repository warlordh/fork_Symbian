Configuring LBS to use the Privacy Notifiers

The LBS Administration setting KLbsSettingPrivacyHandler must be set to CLbsAdmin::EPrivacyHandleByNotifier to configure LBS to use the Privacy Notifiers.

The setting LbsSettingPrivacyTimeoutAction specifies the default response that the LBS subsystem makes if the user does not make a response to a privacy request and it is timed out by the LBS subsystem. This default action can be either to reject the request, to ignore it, or to take the default action that was specified as part of the request. See LBS Administration for details of how to configure the settings either through the LBS Administration API or in the LBS central repository initialisation file.
