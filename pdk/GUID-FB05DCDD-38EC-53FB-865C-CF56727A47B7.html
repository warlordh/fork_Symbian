Multimedia Circuit Switched Call Settings Tutorial

This tutorial describes how to setup and control a multimedia circuit switched (CS) call.

The Multimedia CS calls is a 3G WCDMA specific service that allows two or more media components such as voice, audio, data, video and picture to be combined in a call. A multimedia call is treated by the Mulitmode ETel API as a data call that is used to run the 3G-324M Multimedia protocol over the top of the data connection.


Use RMobileCall::NotifyVoiceFallback() to request a notification if a fallback occurs.
