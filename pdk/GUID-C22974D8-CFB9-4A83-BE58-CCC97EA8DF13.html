Device Driver Writing Guide

This set of documents explains how to write a device driver.

  • Device Driver Writing Guide Quick Start
    Conceptual information that a developer should know before creating a device driver.
  • Introduction
    This set of documents describes the device driver development process and provides links to example material.
  • Minimal Driver Requirements
    This section explains how to make a valid Symbian Platform driver.
  • Design Options
    These documents describe high-level implementation choices for a device driver.
  • The LDD/PDD Model
    This document describes how device drivers are implemented as logical device drivers (LDDs) and physical device drivers (PDDs).
  • Memory Management
    These documents describe how device drivers handle memory buffers and transfer data.
  • User Requests and Synchronisation
    This document introduces user requests and synchronisation methods used by device drivers.
  • Interrupts
    This document describes how device drivers use interrupts.
  • Hardware Registers
    This document describes how device drivers use hardware registers.
