Migration Tutorial: EKA1 to Kernel Architecture 2

Symbian platform v8.1b created a new version of the Kernel Architecture. This replaced the previous Kernel Architecture, called EKA1. This section describes how to change a base port that uses EKA1 to use Kernel Architecture 2.
