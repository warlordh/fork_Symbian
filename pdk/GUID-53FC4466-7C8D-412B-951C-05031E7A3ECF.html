Proximity Monitoring Sensor Channel

The proximity monitoring sensor channel indicates how close the Symbian device is to, for example, the user's hand or ear.

The following table provides information about the proximity monitoring sensor channel:

Header sensrvproximitysensor.h
Data type State
Channel type id KSensrvChannelTypeIdProximityMonitor
Data class TSensrvProximityData
Data units TProximityState Enumeration

The proximity is a binary flag that indicates if the Symbian device is close to or not close to the user. If the proximity cannot be determined accurately then the proximity state is set as undefined.
