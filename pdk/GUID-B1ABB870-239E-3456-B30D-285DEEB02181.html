Constructor & Destructor Documentation

TContextConfigMbmsV1()

IMPORT_CTContextConfigMbmsV1()

Standard constructor. Sets the member data values to the defaults. This class contains member data compatible with parameters required for configuring a MBMS Broadcast context.

The iExtensionId parameter, set automatically by ETel, is used by the TSY when unpacking this class from a TPckg<> to indicate it is to be unpacked as a TContextConfigMbmsV1 class.
