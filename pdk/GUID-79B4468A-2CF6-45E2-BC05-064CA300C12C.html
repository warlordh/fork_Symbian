Using skin support in buttons

Buttons support skins by default. If the default button frames are not suitable, you can change them. Use the methods SetFrameAndCenterIds() and SetBackgroundIds() in the class CAknButton to change the button frame IDs.

For further implementation information on skins, see the Skins API.
