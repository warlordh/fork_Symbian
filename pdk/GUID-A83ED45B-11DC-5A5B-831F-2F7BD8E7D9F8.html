Concepts

This section describes the concepts related to the Comms User Prompt Manager.

  • Comms User Prompt Service
    The Comms User Prompt Service (UPS) enables even unsigned applications to access the network if user authorisation is provided.
  • Policy Files
    The policy files specify the Service ID (SID) and the options to be passed to the user interface using the dialog box. The policy files are resource files, which are compiled into .rsc files. A default policy is required to check the IP connections.
  • Policy Check Requests
    This section describes how the client application requests are authorised by the User Prompt Service.
