HandleGetNetworkProviderNameReqL()

TInt HandleGetNetworkProviderNameReqL()[pure virtual]

The CTSY Dispatcher shall invoke this function on receiving the ECustomGetNetworkProviderNameIPC request from the CTSY.

It is a request call that is completed by invoking CCtsyDispatcherCallback::CallbackPhoneGetNetworkProviderNameComp().

Implementation of this interface should return the name of the current network provider.
