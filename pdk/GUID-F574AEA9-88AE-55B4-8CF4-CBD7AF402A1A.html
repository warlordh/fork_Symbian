Enabling MTP over USB Tutorials

The USB Device Working Group has not defined a standard class and sub-class ID for MTP. To allow USB devices to plug into Windows and handle MTP requests without installing software or requiring user interactions, device vendors must first implement Microsoft OS descriptor (MOD) in their user implementations. MTP over USB can then be enabled in two ways: by using the USB Manager and by writing an MTP client application.

  • Setting Microsoft OS Descriptor
    This section describes how to set the Microsoft OS Descriptor (MOD) for using Symbian MTP over USB services.
  • Enabling MTP over USB
    This section describes two ways to enable MTP over USB, using the USB Manager and writing an MTP client application.
