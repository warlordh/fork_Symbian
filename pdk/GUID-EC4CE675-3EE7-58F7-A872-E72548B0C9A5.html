Sound Device Collection

The Sound Device collection provides a common interface for all audio functionality. Key audio components in the Sound Device collection are the Advanced Audio Adaptation Framework (A3F) DevSound and Audio Component Framework.

  • Sound Device Collection Overview
  • A3F Audio Component Framework Component
    The Advanced Audio Adaptation Framework (A3F) Audio Component Framework component provides a common interface for accessing audio resources, playing tones, and configuring audio for playing and recording.
  • A3F DevSound Component
    The A3F DevSound component provides hardware accelerated access to audio functionality on the Symbian platform.
