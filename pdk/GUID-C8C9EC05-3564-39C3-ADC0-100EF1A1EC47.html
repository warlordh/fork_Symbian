RSat::TLocalInfoV3 Class Reference

class RSat::TLocalInfoV3 : public RSat::TSatPCmdBase

Contains a PROVIDE LOCAL INFORMATION proactive command issued by the UICC. See ETSI TS 102 223 and RSat::NotifyLocalInfoPCmd() for more information about the command.

Request notification for PROVIDE LOCAL INFORMATION with RSat::NotifyLocalInfoPCmd(), passing it an object in which to store the command when it arrives. Pass a packaged version of the latest "TLocalInfo" class. For Symbian OS versions from 7.0s to v8.1a and 8.1b use RSat::TLocalInfoV3Pckg, a packaged RSat::TLocalInfoV3. For later versions of Symbian OS, check for an updated API (RSat::TLocalInfoV4 etc).
