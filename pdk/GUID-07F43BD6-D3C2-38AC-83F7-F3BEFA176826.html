RSat::TSetFramesRspV6 Class Reference

class RSat::TSetFramesRspV6 : public RSat::TSatPCmdBase

Contains your response to a SET FRAMES proactive command. Pass this to the UICC with RSat::TerminalRsp(). See ETSI TS 102 223 and RSat::NotifySetFramesPCmd() for more information about the command.

For Symbian OS version 9.4 pass RSat::TerminalRsp() a RSat::TSetFramesRspV6Pckg (a packaged TSetFramesRspV6). For later versions of Symbian OS, check for an updated API (RSat::TSetFramesRspV7 etc).
