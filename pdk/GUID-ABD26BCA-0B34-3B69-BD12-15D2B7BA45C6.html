TOMASuplPosTechnology2 Class Reference

class TOMASuplPosTechnology2 : public TOMASuplPosTechnology
TOMASuplPosTechnology2 describes the positioning method extension for SUPL v2.0 . Possible values for POS technology are

  • A-GPS SET Assisted only

  • A-GPS SET Based only

  • Autonomous GPS

  • AFLT (Advanced Forward Link Trilateration)

  • EOTD (Enhanced Observed Time Difference)

  • OTDOA (Observed Time Difference of Arrival)
