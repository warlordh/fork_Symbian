Setting up the Test Environment

Before you build and run the tests, you must first setup the test environment for the SysLibs test suite.

  • Device Setup

    The device under test (hardware platform/phone/emulator) should contain an appropriate Symbian platform build or licensee UI build (S60, UIQ, MOAP etc).

  • PC Setup

    The PC must have Symbian or equivalent Developer Kit, PDT and relevant compilers to build the test code (code warrior, rvct, gcce).
