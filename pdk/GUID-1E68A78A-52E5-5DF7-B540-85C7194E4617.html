Graphics Device Interface Tutorials

This section provides examples of working with the GDI API to perform various drawing operations, to store and restore pictures and to zoom.

  • Drawing to a Graphics Context Tutorials
  • Picture Tutorials
    This section provides examples of working with pictures. In this context a picture is an object that can be drawn to a graphics context and can be stored and restored.
  • Twips and Zooming Tutorials
    This section provides examples that demonstrate the effects of different zoom factors on text and graphics.
