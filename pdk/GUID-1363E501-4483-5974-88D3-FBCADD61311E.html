Multimode Call

RMobileCall encapsulates information and functionality for a particular call. It is derived from the RCall interface of core telephony

The functionality of RCall is divided into a number of functional units, for the purpose of easier documentation and navigation and selectable support by TSY. Some RCall functions must be supported by the TSY, while others are optional and only need to be supported if the TSY supports that functional area.
