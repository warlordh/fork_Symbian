RSat::TOpenCsChannelV2 Class Reference

class RSat::TOpenCsChannelV2 : public RSat::TOpenChannelBaseV2

This contains a packet switched OPEN CHANNEL proactive command which asks the phone to open a channel to a circuit switched bearer. All the parameters for the channel are supplied in RSat::TOpenCsChannelV2's fields and its base class, RSat::TOpenChannelBaseV2.

Note: Alternative OPEN CHANNEL commands request a packet switched or local channel; see RSat::TOpenGprsChannelV2 and RSat::TOpenLocalLinksChannelV2 respectively.

More description of the OPEN CHANNEL command can be found in RSat::NotifyOpenChannelPCmd().
