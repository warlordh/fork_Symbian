RSat::TSetUpCallRspV2 Class Reference

class RSat::TSetUpCallRspV2 : public RSat::TSetUpCallRspV1

Contains your response to a SET UP CALL proactive command. Pass this to the UICC with RSat::TerminalRsp(). See ETSI TS 102 223 and RSat::NotifySetUpCallPCmd() for more information about the command.

For Symbian OS v6.1 or v7.0 pass RSat::TerminalRsp() a RSat::TSetUpCallRspV1Pckg (a packaged TSetUpCallRspV1). For versions up to and included v8.1a and v8.1b use RSat::TSetUpCallRspV2Pckg (a packaged TSetUpCallRspV2). For later versions of Symbian OS, check for an updated API (RSat::TSetUpCallRspV3 etc).
