CMceTimerManager Class Reference

class CMceTimerManager : public CBase

CMceTimerManager is the main class of the Lightweight timer subsystem. It is intended to be used in a way that there is only one instance of the CMceTimerManager class, and the basic timer services are available through the MTimerManager interface.

Several logical timers can run simultaneously, but only one CTimer derived timer is used.
