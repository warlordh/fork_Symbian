RSat::TOpenCsChannelV5 Class Reference

class RSat::TOpenCsChannelV5 : public RSat::TOpenCsChannelV2

This contains a packet switched OPEN CHANNEL proactive command which asks the phone to open a channel to a circuit switched bearer. All the parameters for the channel are supplied in RSat::TOpenCsChannelV5's fields and its base classes, RSat::TOpenCsChannelV2 and RSat::TOpenChannelBaseV2.

Note: Alternative OPEN CHANNEL commands request a packet switched or local channel; see RSat::TOpenGprsChannelV5 and RSat::TOpenLocalLinksChannelV5 respectively.

More description of the OPEN CHANNEL command can be found in RSat::NotifyOpenChannelPCmd() and ETSI TS 102 223.
