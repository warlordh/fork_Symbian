Network Adaptation

The Network Adaptation collection contains Protocol Module ECOM plug-ins that can be used by the Network Gateway to support location and privacy requests.

  • Network Adaptation Overview
    The Network Adaptation collection contains Protocol Module ECOM plug-ins that can be loaded by the Network Gateway to support location based services privacy requests and location requests.
  • Privacy Protocol Module
    The Privacy Protocol Module (PPM) is an optional LBS component that allows the LBS subsystem to process privacy requests from the device domestic OS. The privacy request may have been received from the network or from an installed application.
  • SUPL Protocol Module
    The LBS subsystem contains a reference Network Protocol Module that supports the Open Mobile Alliance (OMA) SUPL V1.0 standard. This section contains documentation that describes the Symbian platform reference SUPL Protocol Module, its supporting components and APIs, its runtime behaviour and details about how it is configured by device creators.
