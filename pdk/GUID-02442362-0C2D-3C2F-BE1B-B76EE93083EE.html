RSat::TSendDataRspV2 Class Reference

class RSat::TSendDataRspV2 : public RSat::TSatPCmdBase

Contains your response to a SEND DATA proactive command. Pass this to the UICC with RSat::TerminalRsp(). See ETSI TS 102 223 and RSat::NotifySendDataPCmd() for more information about the command.

For Symbian OS versions from 7.0s to v8.1a and v8.1b pass RSat::TerminalRsp() a RSat::TSendDataRspV2Pckg (a packaged TSendDataRspV2). For later versions of Symbian OS, check for an updated API (RSat::TSendDataRspV3 etc).
