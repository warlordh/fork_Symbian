RSat::TCloseChannelV2 Class Reference

class RSat::TCloseChannelV2 : public RSat::TSatPCmdBase

This contains a CLOSE CHANNEL proactive command which asks the phone to close a channel previously open with an OPEN CHANNEL command. See ETSI TS 102 223 and RSat::NotifyCloseChannelPCmd() for more information about the command. Request notification for CLOSE CHANNEL with RSat::NotifyCloseChannelPCmd(), passing it an object in which to store the command when it arrives. Pass a packaged version of the latest "TCloseChannel" class. For Symbian OS versions from 7.0s to v8.1a and 8.1b use RSat::TCloseChannelV2Pckg, a packaged RSat::TCloseChannelV2. For later versions of Symbian OS, check for an updated API (RSat::TCloseChannelV3 etc).
