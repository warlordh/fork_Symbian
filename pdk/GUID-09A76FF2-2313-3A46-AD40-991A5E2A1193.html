const TInt KCoreHciInterfaceUid

const TIntKCoreHciInterfaceUid

The UID of the Core HCI API. If this API ever has to change, a new UID and associated M- class will be created. New implementations of the Core HCI may implement the new API. Old (non-updated) Core HCI classes will still work as long as the Stack supports the old API.
