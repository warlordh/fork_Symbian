ESock::CSubConnectionProviderFactoryBase Class Reference

class ESock::CSubConnectionProviderFactoryBase : public ESock::CCommsFactoryBase

Base class for all sub-connection provider factories. Specifies the interface that all sub-connection provider factories must implement. It also providers the ECOM framework (via CCommsFactoryBase) to load sub-connection provider factories from other Dynamic Linked Libraries.

Since 9.0
