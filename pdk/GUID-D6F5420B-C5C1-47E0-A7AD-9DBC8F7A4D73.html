EGL Extensions

This section describes the supported EGL extensions and explains their expected behavior on the Symbian platform.

  • EGL Reusable Sync Extension
    This topic describes the EGL reusable sync extension, which introduces sync objects into EGL. Sync objects provide a synchronization mechanism and represent events, whose completion can be tested or waited on.
  • EGL Resource Profiling Extension
    This topic describes the EGL resource profiling extension, which provides a way to retrieve information about GPU resource utilization. UI middleware can use this extension to find out how much memory is being used by the objects created by EGL.
