Multimedia Guide

Find everything you need to know about the Symbian platform Multimedia Subsystem in this section.

  • Multimedia API Tests
  • Multimedia APIs and Frameworks Collection
  • HDMI
    This section describes HDMI functionality within the Multimedia subsystem.
  • Multimedia Device Framework Collection
  • Multimedia Hardware Adaptation Interfaces Collection
    The Multimedia Hardware Adaptation Interfaces (HAI) collection is an interface to control video components.
  • Multimedia Plug-ins Collection
    The Multimedia Plug-ins collection facilitates a plug-in framework to handle the Multimedia audio, video and image data.
  • Multimedia Software Adaptation Collection
    The Multimedia Software Adaptation Collection contains one component, the Video Renderer, which enables video rendering to a graphics surface.
  • Multimedia Validation Suite (MVS) User Guide
    This section describes the Multimedia Validation Suite (MVS), and how to use it to test your newly developed audio and video controller and codec plug-ins:
  • OpenMAX IL Collection
    The OpenMAX Integration Layer (IL) collection is an open standard developed by The Khronos Group. The OpenMAX IL serves as a low-level interface for audio, video and imaging components used in mobile devices.
  • Sound Device Collection
    The Sound Device collection provides a common interface for all audio functionality. Key audio components in the Sound Device collection are the Advanced Audio Adaptation Framework (A3F) DevSound and Audio Component Framework.
