Requesting Current Bearer Identifier

This tutorial explains how to request current bearer identifier.

Procedure

  1. Obtain the TMsvId (Service ID) for the connected server MTM.

  2. Call CMsvSession::ServiceAccessPointId(), passing the service ID and a TUint32 reference.

Result

On return, the TUint32 stores the currently connected bearer identifier.
