Class summary

CFileMan MFileManObserver

Description

Notify demonstrates using MFileManObserver how to observe a file copy operation carried out through the file management class CFileMan. The build process puts a file, dateFile, on the C drive. The program copies this to a new file, dateFileCopy, using MFileManObserver to report the progress of the operation.
