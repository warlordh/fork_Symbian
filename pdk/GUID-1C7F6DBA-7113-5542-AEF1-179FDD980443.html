SWI Analysis ToolKit Reference

This section provides detailed information about the syntax and usage of the tools available in the SWI Analysis ToolKit.

  • DumpChainValidityTool
    DumpChainValidityTool validates the signing certificate-chains associated with a SIS file and displays a unified list of capabilities that are supported by the validated chains.
  • DumpInstallFileStatusTool
    DumpInstallFileStatusTool checks the capabilities of executables within a SIS file against the capabilities supported by the signing certificates and displays the missing capabilities corresponding to each executable.
  • DumpSWIRegistryTool
    DumpSWIRegistryTool interprets the software install registry entry (.reg file) and displays its content.
  • DumpSWICertstoreTool
    DumpSWICertstoreTool extracts certificate related information such as capabilities and mandatory status from one or more SWI certificate stores.
