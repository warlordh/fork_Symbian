SetFirstServiceHintByte(const TUint8)

IMPORT_C voidSetFirstServiceHintByte(const TUint8aFirstServiceHintByte)

Sets the first service hint byte for the remote device. The service hint bytes indicate the level of IrLMP support provided by the device.

Parameters

const TUint8 aFirstServiceHintByteThe first service hint byte. The individual bits have the following meaning: Bit 0 (0x01) - PnP Compatible Bit 1 (0x02) - PDA/Palmtop Bit 2 (0x04) - Computer Bit 3 (0x08) - Printer Bit 4 (0x10) - Modem Bit 5 (0x20) - Fax Bit 6 (0x40) - LAN Access Bit 7 (0x80) - Extension
