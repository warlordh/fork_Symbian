CTactileAreaRegistry::TTactileAreaChunk Struct Reference

struct CTactileAreaRegistry::TTactileAreaChunk

Items of this type are stored into member variable iChunkArray. There is one item in the array for each connecting application.

Stored window group id is the identifier of the main window group of connecting application. Stored connection handle is the window server connection handle of the application.
