Component Shutdown Aware

The State Aware Session is a library (dll) that provides an API that allows clients to manage notification of system state transitions. Using the API clients can register for notification of system state transition. The client can supply a request status object to the notification request.

    This document explains the step by step procedure to make a component shutdown aware.
  • Registering for Shutdown Notification
    This document explains how to register for shutdown notification for a component.
  • Managing Shutdown
    This document explains how to handle and acknowledge shutdown notification and close the session.
