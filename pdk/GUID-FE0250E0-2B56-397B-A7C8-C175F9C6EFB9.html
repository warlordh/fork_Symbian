HandleStoreInitialiseReqL()

TInt HandleStoreInitialiseReqL()[pure virtual]

The CTSY Dispatcher shall invoke this function on receiving the EMmTsyPhoneBookStoreInitIPC request from the CTSY.

It is a request call that is completed by invoking CCtsyDispatcherCallback::CallbackPhonebookStoreInitialiseComp()

Implementation of this interface should request to initialise the phonebook store, the callback should be completed when store initialisation is completed.
