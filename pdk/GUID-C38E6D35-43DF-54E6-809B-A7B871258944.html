Application Framework Overview

The Application Framework defines a framework to register and manage applications.

Purpose

The Application Framework enables native and non-native applications to run on Symbian devices.

Components

The Application Framework collection consists of the following components:

  • Application Architecture, which provides access to

    • a cached list of applications, and their properties, on the device.

    • various mechanisms for launching applications and switching between tasks.

    For more information on Application Architecture, refer to Application Architecture Overview.

  • View server, which is a view-based framework, enables UI applications to have multiple views and supports switching between the views.

For more information on View Server, refer to View Server Overview.
