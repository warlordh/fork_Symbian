CAppMngr2PackageInfo Class Reference

class CAppMngr2PackageInfo : public CAppMngr2InfoBase

CAppMngr2PackageInfo represents installation package (installation file).

Runtime plug-in must define it's own CAppMngr2PackageInfo derived class to provide the actual functionality for abstract functions defined in CAppMngr2InfoBase class.
