C++ Standard Library (STLport v4)

C++ Standard Library is a Symbian platform port of STLport v4. It enables applications written using Standard C++ to be ported to Symbian platform.

Notes:

  • It is recommended that you use the latest version of Standard C++ Library (STLport v5) to take advantage of it's latest features. For more information about STLport, see http://www.stlport.org/. For information about how to migrate to STLport v5, see Migrating from STLport v4 to v5.

  • For more information about Open C++ (C++ Standard Library - STLport v4) API reference and examples, see Forum Nokia Library.

  • Copyright Acknowledgments for Standard C++ (STLport v4) and Boost Libraries
  • Getting started with C++ Standard Library
    The Standard C++ library depends on P.I.P.S.. The user must have the P.I.P.S. components installed before using the library.
