Playing media files

You can use the MVS to play audio and video media files.

To play a media file:

  1. In the File menu, click Open for Play. The Open file dialog box is displayed.

  2. In the dialog box, if necessary, enter the names of the disk drive and folder containing the file, or select them from the lists.

  3. In the dialog box, enter the file name or select the file from the list.

  4. In the dialog box, click OK to open the file.

  5. Click the tool bar Play button, or in the Controls menu, click Play.

  • To pause a playing audio file, click the tool bar Pause button, or in the Controls menu, click Pause. Repeat this to un-pause the file and resume playing.

    Note: You cannot pause video playback.

  • To stop a file playing, click the tool bar Stop button, or in the Controls menu, click Stop.
