Accessing Large Files using the 64-bit File Server Client API

This tutorial set demonstrates how to access large files using the 64-bit File Server client API.

Before you start, you must:

  • understand the basics of File Services in Symbian Platform

  • be familiar with the File Server Client Side API (see File Server Client Side Overview).

This tutorial set describes the API and behaviour of the File Server Client when dealing with large files. It also explains the modifications that must be made to existing applications to allow them to access large files.
