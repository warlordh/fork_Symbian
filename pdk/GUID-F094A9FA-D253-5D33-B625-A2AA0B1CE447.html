EGL Implementer's Guide

This section provides information for EGL implementers working with the Symbian platform.

Target audience: Device creators.

  • EGL Porting Guide
    This is a guide for implementers of EGL working with the Symbian platform.
  • EGL Multithreading Requirements
    This topic provides EGL implementers with information about the EGL multithreading requirements on the Symbian platform.
  • Window Surface Implementation
    This topic provides an introduction to implementing EGL window surfaces on the Symbian platform. These are used for onscreen rendering into windows that are provided by the Symbian platform (rather than EGL). The details of how to implement window surfaces depend on whether the ScreenPlay variant is in use.
  • Pixmap Implementation
    This topic provides an introduction to implementing EGL pixmap surfaces on the Symbian platform.
