File-Based Certificate and Key Stores

File-Based Certificate and Key Stores manage cryptography certificates, keys and certificate applications in a device. The Symbian platform provides device creators with a file-based software plug-in implementation of key, certificate and certificate application stores in the device such that these can be directly used.
