Messaging Application Guide

The Messaging Application module provides APIs for SMS, email, MMS and OBEX messaging. This guide provides detailed information about the Messaging Application module.

The Messaging Application module consists of the following collections:

  • Mobile Messaging
  • Messaging Application Base
  • MMS Engine
  • Email
