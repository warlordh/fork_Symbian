Location Server

The Location Server handles location requests from installed applications.

  • Location Server Overview
    The Location Server component of the LBS system model contains a Symbian platform server to manage location requests from installed applications (MO-LR self locate requests).
  • LBS Quality Profiles
    A quality profile defines the minimum acceptable horizontal and vertical accuracy of a location fix and the maximum acceptable time to obtain the fix.
