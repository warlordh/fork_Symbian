ImageProcessor::Plugin::MEffectSharpness Class Reference

class ImageProcessor::Plugin::MEffectSharpness : public ImageProcessor::Plugin::MEffect

Plugin API for the sharpness photoart effect. The TEffectSharpness uses this interface class to get access to the image prosessor plugin effect. Once this has been constructed, calls to method functions of TEffectSharpness are passed verbatim to this interface. For further description of required functionality, see TEffectSharpness.
