Deleting the Transport Buffer Size Specifications for Bearer Types

This tutorial explains how to delete the transport buffer size specifications or to revert to default parameters for all bearer types.

Procedure

  1. Create an instance of the CEmailAccounts class.

  2. Use the CEmailAccounts::DeleteImapTransportBufferSizesL() function to delete the transport buffer size specifications.
