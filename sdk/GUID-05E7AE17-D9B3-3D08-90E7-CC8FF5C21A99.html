PackLC(TTaskInfo &, HBufC *&)

IMPORT_C void PackLC ( TTaskInfo & aTask,
HBufC *& aDes
) const

Packs the object into a 16-bit descriptor.

The function then attempts to copy the descriptor into aTask.iName, if aTask.iName is large enough; otherwise, it copies it into aDes (a new HBufC).

aDes is placed on the cleanup stack.

Parameters

TTaskInfo & aTask Task information for this message.
HBufC *& aDes On return, a new descriptor, possibly packaging the object. aDes.Length() == 0 if the object was successfully packed in aTask.iName instead.
