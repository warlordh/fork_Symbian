ImageProcessor::Plugin::MEffectCartoon Class Reference

class ImageProcessor::Plugin::MEffectCartoon : public ImageProcessor::Plugin::MEffect

Plugin API for the cartoon photoart effect. The TEffectCartoon uses this interface class to get access to the image prosessor plugin effect. Once this has been constructed, calls to method functions of TEffectCartoon are passed verbatim to this interface. For further description of required functionality, see TEffectCartoon .
