)

Constructs a new IAS query taking the class name, the attribute and the remote device address.

Parameters

const TDesC8 & aClass The IAS query class name.
const TDesC8 & aAttribute The IAS query attribute.
TUint aRemoteDevAddr The 32 bit address of the remote device to be queried. This is needed because the underlying IrLAP connection may not yet have been established.

TIASQuery()

IMPORT_C TIASQuery ( )

Constructs an empty IAS query object.

The class name, the attribute and the remote device name must be set before this query object can be used.
