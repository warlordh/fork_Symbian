RSat::TPlayToneV6 Class Reference

class RSat::TPlayToneV6 : public RSat::TPlayToneV5

Contains a PLAY TONE proactive command issued by the UICC. See ETSI TS 102 223 and RSat::NotifyPlayTonePCmd() for more information about the command.

Request notification for PLAY TONE with RSat::NotifyPlayTonePCmd() , passing it an object in which to store the command when it arrives. Pass a packaged version of the latest "TPlayTone" class. For Symbian OS 7.0s, 8.0 and 8.1 use RSat::TPlayToneV2Pckg , a packaged RSat::TPlayToneV2 . For Symbian OS 9.2, use RSat::TPlayToneV5Pckg . Use RSat::TPlayToneV1 with older versions of Symbian OS. For Symbian OS 9.4, use RSat::TPlayToneV6Pckg . For later versions of Symbian OS, check for an updated API (RSat::TPlayToneV7 etc).
