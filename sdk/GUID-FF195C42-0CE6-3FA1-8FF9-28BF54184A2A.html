) [static]

Create a CAsyncMac instance (for hardware based MAC plug-in dll implementation)

leave
KErrNone if successful; otherwise, leaves with a system wide error code.

Parameters

CAsyncMac *& aMac The pointer to CMac. This will be initialised with the plug-in implementation of the desired MAC algorithm.
const TUid aAlgorithmUid The specific MAC algorithm desired for evaluation of MAC value. e.g. MD2, SHA1 or AES-XCBC-MAC-96, AES-XCBC-PRF-128
const CKey & aKey Symmetric key for calculating message authentication code value.
const CCryptoParams * aAlgorithmParams The parameters those are specific to a particular MAC algorithm. This is for extendibility and will normally be null.
