TLbtSecurityPolicy Class Reference

class TLbtSecurityPolicy

This class provides the inteface to define security policy. As of now the security policy only provides for security check based on secure id. This class will be extended any future requirements for security checking

The inteface provides the following functionality,

  • Define policy for a particular process based on secure id
