The CTSY Dispatcher shall invoke this function on receiving the EMmTsySimGetICCType request from the CTSY.

It is a request call that is completed by invoking CCtsyDispatcherCallback::CallbackSimGetActiveIccApplicationTypeComp()

This handler is called by the CTSY to retrieve the current active ICC application type. Types are defined in MLtsyDispatchSimGetActiveIccApplicationType::TIccType .
