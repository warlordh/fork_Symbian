RSat::TUssdDataDownloadV6 Class Reference

class RSat::TUssdDataDownloadV6 : private RSat::TSatBase

This data structure contains the details of the USSD Data Download ENVELOPE command issued to the UICC. See ETSI TS 102 223 and RSat::UssdDataDownload() for more information about the command.

Pass a packaged version of the latest "TUssdDataDownloadV6" class. For Symbian OS version 9.4 use RSat::TUssdDataDownloadV6Pckg , a packaged version of RSat::TUssdDataDownloadV6 . For later versions of Symbian OS, check for an updated API (RSat::TUssdDataDownloadV7 etc).
