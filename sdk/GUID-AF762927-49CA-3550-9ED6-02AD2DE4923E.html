DrawArc(const TRect &, TInt, TInt)

void DrawArc ( const TRect & aRect,
TInt aStartAngle,
TInt aArcAngle
) [pure virtual]

Draws the outline of a circular or elliptical arc within the specified area. The resulting arc begins at aStartAngle and extends for ArcAngle degrees, using the current colour. Angles are interpreted such that 0 degrees is at the 3 o'clock position. A positive value indicates a counter-clockwise rotation while a negative value indicates a clockwise rotation.

Parameters

const TRect & aRect The bounding rectangle of the ellipse the arc is part of.
TInt aStartAngle The arc's start angle, in degrees
TInt aArcAngle The arc's aperture, in degrees
