Category: FSCLIENT

Panics with this category are raised on clients when a file server operation or a file server plug-in operation leaves. The leave is trapped, and the panic is raised with the FSCLIENT category; the leave number is used as the panic number.

Note that the leave number is one of the standard leave codes defined by the Symbian platform.
