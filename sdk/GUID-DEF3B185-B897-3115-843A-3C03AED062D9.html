struct @52 component_list

Logger could store tracing information from different components to a single file or create separate logging file for each component (to log to single file enable J_LOG_ALL_LOGS_TO_SINGLE_FILE macro).

This structure contains name of logging file for each component and component's nickname which will be used when all logging info is stored in a single file.
