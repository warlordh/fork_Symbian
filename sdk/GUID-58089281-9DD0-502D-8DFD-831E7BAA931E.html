Reference

This section contains the reference material that applications and tools require to use the Comms Database.

  • C++ API items
    This section lists the C++ API items that make the CommsDat API. Tools and applications use these API items to access tables, records ad fields in the Comms Database.
  • Tables for Application Developers and Phone Creators
    This section lists the public tables that Symbian platform provides. These tables are available to application developers and phone creators.
  • Tables for Phone Creators
    This section lists the tables that Symbian platform distributes. These tables are available to phone creators only.
