Bluetooth Collection

The Bluetooth collection contains the SDP, GAVDP and Notifiers Support components.

  • Bluetooth Collection Overview
    The Bluetooth collection contains the SDP, GAVDP and notifier support components.
  • Bluetooth SDP
    The Bluetooth Service Discovery Protocol (SDP) component enables you to register services to the SDP database and query the database for available services.
  • Bluetooth Notifiers Support
    The Bluetooth Notifiers Support component enables you to create dialogs to interact with device users about Bluetooth interactions.
