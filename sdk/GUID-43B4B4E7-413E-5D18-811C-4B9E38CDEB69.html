PKG File Format

The PKG file format is described in modified BNF notation and must include the following definitions:

  • Languages Supported
  • Language Codes
  • Package-Header
  • Vendor
  • Logo
    Specifies logos corresponding to each language supported by the Symbian device.
  • Package-Body
  • Dependency
  • PKG Properties
  • Package File Examples
