TUint8 iTxBlockAckUsage

A bit-map containing the block ACK usage status for the tx direction. Each bit corresponds to a TID, bit 0 corresponding to TID 0. If a bit is set, block ACK can be used for the corresponding TID. The WLAN device may initiate block ACK session with the AP for the TID in question. If a bit is not set and there already is a block ACK session with the AP for this TID, the WLAN device should terminate the session.
