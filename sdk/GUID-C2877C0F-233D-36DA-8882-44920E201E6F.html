TInt HandleRejectNetworkInitiatedContextActivationRequestReqL ( ) [pure virtual]

The CTSY Dispatcher shall invoke this function on receiving the EPacketRejectActivationRequest request from the CTSY.

It is a request call that is completed by invoking CCtsyDispatcherCallback::CallbackPacketServicesRejectNetworkInitiatedContextActivationRequestComp()

Implementation of this interface should reject the network initiated context activation request.
