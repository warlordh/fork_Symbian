BitGDI Component

The BitGDI component provides extensions of the CGraphicsDevice and CGraphicsContext abstract base classes that are provided by the GDI component. These extensions enable hardware-independent access to screen attributes and drawing to bitmapped devices and screens independent of the display. BitGDI is used by the Window Server in the non-ScreenPlay variant and legacy applications.

The CBR package name for the BitGDI component is graphics_bitgdi .
