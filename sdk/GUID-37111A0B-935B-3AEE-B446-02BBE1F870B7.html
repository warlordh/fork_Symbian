TDecryptFilter Class Reference

class TDecryptFilter : public TSecureFilter

A decrypting filter.

An object of this type is used by RDecryptStream to decrypt stream data as it is read from a source stream.

Decryption itself is performed by an instance of a class implementing the CPBDecryptor interface.
