RSat::TPerformCardApduV2 Class Reference

class RSat::TPerformCardApduV2 : public RSat::TSatPCmdBase

Contains a PERFORM CARD APDU proactive command issued by the UICC. See ETSI TS 102 223 and RSat::NotifyPerformCardApduPCmd() for more information about the command.

Request notification for PERFORM CARD APDU with RSat::NotifyPerformCardApduPCmd() , passing it an object in which to store the command when it arrives. Pass a packaged version of the latest "TPerformCardApdu" class. For Symbian OS version from 7.0s to v8.1a and 8.1b use RSat::TPerformCardApduV2Pckg , a packaged RSat::TPerformCardApduV2 . For later versions of Symbian OS, check for an updated API (RSat::TPerformCardApduV3 etc).
