Swi::Sis::CProperty Class Reference

class Swi::Sis::CProperty : public Swi::Sis::CField

This class represents a SisProperty. SisProperty is a basic structure found in SIS files. Each property represented by this object is stored as a (key,value) pair. Both key and value are 32 bits integers.

For a full description see SGL.GT0188.251.
