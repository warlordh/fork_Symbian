HandleSelectNetworkCancelReqL()

TInt HandleSelectNetworkCancelReqL ( ) [pure virtual]

The CTSY Dispatcher shall invoke this function on receiving the EMobilePhoneSelectNetworkCancel request from the CTSY.

It is a request call that is completed by invoking CCtsyDispatcherCallback::CallbackPhoneSelectNetworkCancelComp()

Implementation of this interface should cancel a pending network selection.
