DirectGdi::TTextParameters Class Reference

class DirectGdi::TTextParameters

Parameters used in drawing text within supplied context. It is used by CGraphicsContext::DrawText() and CGraphicsContext::DrawTextVertical() family of APIs to draw text from iStart to iEnd within the supplied text descriptor.
