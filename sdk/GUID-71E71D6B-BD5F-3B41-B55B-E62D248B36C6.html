TContextConfigR99_R4()

IMPORT_C TContextConfigR99_R4 ( )

Standard constructor. Sets the member data values to the defaults.

This class contains member data compatible with parameters required for configuring a context on the R99/R4 packet networks. The iExtensionId parameter, set automatically by ETel, is used by the TSY when unpacking this class from a TPckg<> to indicate it is to be unpacked as a TContextConfigR99_R4 class.
