Key comparison

This document describes how text and descriptor keys can be compared.

Text and descriptor keys can be compared in one of three ways:

  • keys are folded for the purpose of comparison

  • keys are compared using the collation rules for the current locale

  • keys are compared normally.

Numeric keys are compared arithmetically in the usual way.
