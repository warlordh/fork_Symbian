RSat::TPowerOnCardV2 Class Reference

class RSat::TPowerOnCardV2 : public RSat::TSatPCmdBase

Contains aPOWER ON CARD proactive command issued by the UICC. See ETSI TS 102 223 and RSat::NotifyPowerOnCardPCmd() for more information about the command.

Request notification for POWER ON CARD with RSat::NotifyPerformCardApduPCmd() , passing it an object in which to store the command when it arrives. Pass a packaged version of the latest "TPowerOnCard" class. For Symbian OS versions from 7.0s to v8.1a and 8.1b use RSat::TPowerOnCardV2Pckg , a packaged RSat::TPowerOnCardV2 . For later versions of Symbian OS, check for an updated API (RSat::TPowerOnCardV3 etc).
