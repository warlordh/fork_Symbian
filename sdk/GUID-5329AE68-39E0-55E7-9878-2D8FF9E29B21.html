User Prompt Service

User Prompt Service (UPS) allows device creators to choose whether security decisions (for example, allowing an application to make a network connection) are processed silently or require the user to be prompted. This behaviour can be configured statically using resource files known as UPS policy files or, dynamically using ECom plug-ins loaded into the UPS server.
