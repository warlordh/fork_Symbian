TBool disableDynamicPowerModeManagement

TBool disableDynamicPowerModeManagement

This setting is relevant only if powerMode is EPowerModePs. If ETrue, disables the dynamic power mode management handling in UMAC. If EFalse, allows the dynamic power mode management handling in UMAC.
