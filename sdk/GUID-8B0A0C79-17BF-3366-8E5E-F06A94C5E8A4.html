class MBrCtlSoftkeysObserver

The MBrCtlSoftkeysObserver class handles requests to change the softkeys.

Requests the host application to change a softkey. The host application may or may not change the softkey.
Since
2.8

Parameters

TBrCtlKeySoftkey aKeySoftkey Determines whether to update the left softkey or the right softkey. Values: EKeyRight to update the right softkey EKeyLeft to update the left softkey
const TDesC & aLabel The label associated with the softkey update
TUint32 aCommandId The command to use if the softkey is selected by the user
TBrCtlSoftkeyChangeReason aBrCtlSoftkeyChangeReason Reason for the softkey change Value: EChangeReasonIdle (The other values are reserved for future use.)
