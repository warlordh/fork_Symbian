RSubFrame1ReservedArrayReader Class Reference

class RSubFrame1ReservedArrayReader : public RDataReaderArray< TUint >

SubFrame1Reserved Array Reader class. These are reserved bits in the subframe 1 of the GPS navigation message - an array of 4 elements (23, 24, 24 and 16 bits in size). Defined using RRC spec.
