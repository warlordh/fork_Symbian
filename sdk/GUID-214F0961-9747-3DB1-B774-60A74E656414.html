CSsmLoadAmaStart Class Reference

class CSsmLoadAmaStart : public CBase

Shim dll used to remove the dependency of loadamastart.dll which is common symbian component on the amastart.dll which is optional replaceable. loadamastart.dll will load amastart.dll.

See CLoadAmaStart in the sysstart component.
