HandleSimRefreshDoneReqL(TInt)

TInt HandleSimRefreshDoneReqL ( TInt aError ) [pure virtual]

The CTSY Dispatcher shall invoke this function on receiving the EMmTsySimRefreshDoneIPC request from the CTSY.

It is a request call that has no complete.

Implementation of this interface indicates to the LTSY the SIM refresh has finished.

Parameters

TInt aError An error code relating to the result of the refresh.
