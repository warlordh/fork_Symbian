RMobilePhone::TMobilePhoneServiceProviderNameV2 Class Reference

class RMobilePhone::TMobilePhoneServiceProviderNameV2 : public RMobilePhone::TMultimodeType

Used to retrieve the Service Provider Name from the ICC. The SP is actually the entity that provides the service. The SP could have contractual agreements with several home PLMN that the ME can register to. The display preference of the registered PLMN and/or the SPN on the ME can be defined on the UICC.
