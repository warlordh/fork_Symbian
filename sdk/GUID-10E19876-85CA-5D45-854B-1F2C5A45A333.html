Overlay

This tutorial introduces you to the overlay functionality of the Ecam component.

Purpose

This document describes the various APIs used with overlay.

Introduction

Overlay is an image which is superimposed on top of the main image with a required transparency. The main image can be a single image during image capture mode (still image/ burst/ bracket), video image frame, snapshot image, or image frames for viewfinder (direct or client-based).

The client can use it to overlay an image onto the viewfinder, snapshots, still images and video. A client can create multiple overlays, where each overlay can be in a different format. The properties of an overlay can be changed after it has been created.
