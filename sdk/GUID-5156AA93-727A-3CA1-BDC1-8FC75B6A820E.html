TUint8 iLowerRefreshRateSupported

TUint8 iLowerRefreshRateSupported

If mode has two refresh rates. If is True (1), then lower frequency is supported. If mode has only one refresh rate, then this has to be True (1). E.g. CEA mode 1: 640x480p .94/60Hz, so this defines is 59.94Hz supported (lower) or not. Don't care if mode has only one refresh rate (e.g. CEA 17 720x576p ) iLowerRefreshRateSupported=1=Lower refresrate supported (e.g. 59.94Hz). iHigherRefreshRateSupported=1=Higher refreshrate supported (e.g. 60Hz)
