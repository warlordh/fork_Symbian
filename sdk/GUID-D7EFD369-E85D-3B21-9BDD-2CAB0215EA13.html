TOstArray Class Reference

class TOstArray

Template class for array parameter types For example, to wrap an integer array to a trace: TInt arr[5]; OstTraceExt( GRP, TRC, "Array: %{int32[]}", TOstArray< TInt >( arr, 5 ) );
