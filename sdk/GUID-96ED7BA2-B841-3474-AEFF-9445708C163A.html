RSat::TSendUssdRspV1 Class Reference

class RSat::TSendUssdRspV1 : public RSat::TSatPCmdBase

Contains your response to a SEND USSD proactive command. Pass this to the UICC with RSat::TerminalRsp() . See ETSI TS 102 223 and RSat::NotifySendUssdPCmd() for more information about the command.

For Symbian OS versions up to and including v8.1a and v8.1b pass RSat::TerminalRsp() a RSat::TSendUssdRspV1Pckg (a packaged TSendUssdRspV1 ). For later versions of Symbian OS, check for an updated API (RSat::TSendUssdRspV2 etc).
