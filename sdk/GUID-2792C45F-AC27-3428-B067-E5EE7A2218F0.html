RSat::TSendDtmfRspV1 Class Reference

class RSat::TSendDtmfRspV1 : public RSat::TSatPCmdBase

Contains your response to a SEND DTMF proactive command. Pass this to the UICC with RSat::TerminalRsp() . See ETSI TS 102 223 and RSat::NotifySendDtmfPCmd() for more information about the command.

For Symbian OS versions up to and including v8.1a and v8.1b pass RSat::TerminalRsp() a RSat::TSendDtmfRspV1Pckg (a packaged TSendDtmfRspV1 ). For later versions of Symbian OS, check for an updated API (RSat::TSendDtmfRspV2 etc).
