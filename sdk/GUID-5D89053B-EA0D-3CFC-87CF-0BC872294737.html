TInt HandlePacketAttachReqL ( ) [pure virtual]

The CTSY Dispatcher shall invoke this function on receiving the EPacketAttach request from the CTSY.

It is a request call that is completed by invoking CCtsyDispatcherCallback::CallbackPacketServicesPacketAttachComp()

Implementation of this interface should issue an attach request to the LTSY.
