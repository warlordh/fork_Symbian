OS Security Guide

OS Security provides the fundamental services to make protocols and services in the Symbian platform secure. It provides frameworks and interfaces that are required to fulfill the demands of Symbian Platform Security Architecture. It includes collections of components that have to be modified or redesigned to support Platform Security.

  • OS Security Quick Start
    OS Security on the Symbian platform enables data confidentiality, integrity and authentication. It provides high-level services, such as key and certificate management, authentication, secure user prompting, and Digital Rights Management (DRM) framework.
