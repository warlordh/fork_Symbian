RSat::TLanguageNotificationRspV2 Class Reference

class RSat::TLanguageNotificationRspV2 : public RSat::TSatPCmdBase

Contains your response to a LANGUAGE NITIFICATION proactive command. Pass this to the UICC with RSat::TerminalRsp() . See ETSI TS 102 223 and RSat::NotifyLanguageNotificationPCmd() for more information about the command.

For Symbian OS versions from v7.0s to v8.1a and v8.1b, pass RSat::TerminalRsp() a RSat::TLanguageNotificationRspV2Pckg (a packaged TLanguageNotificationRspV2 ). For later versions of Symbian OS, check for an updated API (RSat::TLanguageNotificationRspV3 etc).
