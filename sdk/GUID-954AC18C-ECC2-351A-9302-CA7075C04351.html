CEmailExtension Class Reference

class CEmailExtension : public CBase
Base class for extensions. Instances of subclasses are requested from CExtendableEmail::ExtensionL() with specific extension UID. When client has ended using an extension it must be released with CExtendableEmail::ReleaseExtension() .
