ContentAccess::MAgentRightsBase Class Reference

class ContentAccess::MAgentRightsBase

Interface used by agents as a base class for their own rights objects

Agent derived classes will contain a complete object capable of describing rights for content managed by that agent.

All derived classes must implement the serialization functions InternalizeL() and ExternalizeL()
